##Automatický krmič
Zařízení složí k pravidelnému dávkování krmiva.
SYstém kombinuje automatický režim s možností ručního ovládání pomocí tlačítek. Uživatel může ovlivnit chování zařízení podle aktuální potřeby. Během aktivního dávkování je chod signalizován LED diodou.
Projekt je navržen jako základ který lze dál upravovat a rozšiřovat podle konkrétních požadavků.
