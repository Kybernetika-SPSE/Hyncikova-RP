<h1>Automatický krmič</h1><br>
<h2>Zařízení složí k pravidelnému dávkování krmiva</h2>.<br>
Systém kombinuje automatický režim s možností ručního ovládání pomocí tlačítek. Uživatel může ovlivnit chování zařízení podle aktuální potřeby. Během aktivního dávkování je chod signalizován LED diodou.<br>
Projekt je navržen jako základ který lze dál upravovat a rozšiřovat podle konkrétních požadavků.<br>
